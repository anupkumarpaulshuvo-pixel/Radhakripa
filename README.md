# 🙏 Radha Naam Jap

A beautiful Progressive Web App for daily Radha Naam Jap practice.

## Features

- **Jap Counter** — tap to count with floating राधा animations
- **Auto Timer** — starts on first tap, pauses after 6s idle
- **28 Names of Radha** — cycle through all 28 names in Bengali
- **Stotram Tracker** — track daily stotram recitation
- **Brahmacharya Tracker** — auto-maintains past days (only log breaks)
- **Jap Calendar** — visualize your practice history
- **Stats** — streaks, weekly/monthly summaries, jap time
- **Targets** — daily & lifetime jap goals with progress rings
- **Brahma Muhurta & Sandhya Kal** — calculated from your GPS location
- **Firebase Sync** — real-time sync across all devices

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your app will be live at `https://yourusername.github.io/repo-name`

## Install as App (PWA)

- **Android**: Open in Chrome → three-dot menu → "Add to Home Screen"
- **iPhone**: Open in Safari → Share button → "Add to Home Screen"

## Firebase Setup

The Firebase config is already included. To use your own project:
1. Go to [console.firebase.google.com](https://console.firebase.google.com)
2. Create a project, enable **Authentication** (Email/Password) and **Firestore**
3. Replace the `firebaseConfig` object in `index.html`
